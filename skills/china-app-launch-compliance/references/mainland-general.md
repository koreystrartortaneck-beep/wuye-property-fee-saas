# 中国大陆通用要求

## 基础决策

| 问题 | 处理方式 |
|---|---|
| 是否在境内提供互联网信息服务 | 是则核验 APP备案；工信部口径包含 APP、小程序、快应用等 |
| 是否已有网站备案 | 仅作为备案资料复用线索，不假设自动完成 APP备案 |
| 是否属于经营性互联网信息服务 | 结合服务内容和收费本质判断是否触发增值电信业务许可；不能因“收物业费”直接判定需要 ICP许可证 |
| 是否含新闻、出版、视听、网络文化、医疗、教育、宗教等内容 | 条件触发前置审批、许可、备案或平台专项材料核验 |
| 是否使用境内域名、服务器或云资源 | 核验域名实名、接入服务、备案信息与实际主办者一致性 |
| 是否达到网络安全等级保护适用条件 | 开展定级、备案和测评适用性判断，不把所有小程序统一写成固定等级 |

## APP备案

- 核验主办者、域名、服务器、服务内容、负责人、平台报备信息和实际运营主体是否一致。
- 分别获取小程序与原生 App 的备案/平台状态；不得用网站 ICP 备案号代替所有移动应用备案证明。
- 备案是基础登记，不等于取得增值电信业务或行业专项许可。
- 产品名称、域名、主体或服务内容变化时，核验变更备案义务。

## 常见主体材料

营业执照、统一社会信用代码、法定代表人/负责人信息、经办授权、联系方式、域名及服务器证明、隐私政策、用户协议、软件著作权或权利来源证明。它们是否“必须”取决于备案渠道和目标平台的实时要求。
