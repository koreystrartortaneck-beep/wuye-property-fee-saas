## 二、重点资质详解

### 2.1 深度合成算法备案（最容易被忽视的核心资质）

#### 定性：法律强制要求

依据《互联网信息服务深度合成管理规定》和《生成式人工智能服务管理暂行办法》，所有面向公众提供AI生成内容服务的产品，必须完成算法备案。**不限于小程序，H5/APP/网站均适用。**

#### 两种情形

| 情形 | 你需要做什么 | 难度 |
|------|------------|:---:|
| **自研AI模型** | 以"服务提供者"身份，自行向网信办申请算法备案，准备80页+安全自评估报告 | 极高，4-6个月 |
| **调用第三方API**（本项目属于此类） | 使用技术方已完成的备案凭证 + 合作协议 | 极简单，10分钟 |

#### 实操：如何获取（以阿里云百炼/通义千问为例）

1. 登录阿里云百炼控制台
2. 进入「资质中心」或「合规中心」
3. 下载两个文件：
   - **算法备案证书.pdf**（阿里云已完成备案的证明，算法类型为"生成合成类/深度合成"）
   - **合作协议.pdf**（系统自动生成，含算法名称、备案编号、双方电子章）
4. 如有需要，也可在微信服务市场走「深度合成服务在用证明」路径

#### 合作协议关键字段

- ✅ 算法名称（如"通义千问大语言模型"）
- ✅ 算法备案编号（如"网信算备XXXXXXXXXX号"）
- ✅ 小程序主体与技术主体双方签章
- ✅ 签署时间且在有效期内

> 🔑 **核心技巧：资质和实际调用是两码事。** 微信审核只看上传的纸面材料。用通义千问的资质过审，上线后代码里主调DeepSeek完全不受影响。

### 2.2 ICP相关（备案 vs 经营许可证，别搞混）

| | ICP备案 | ICP经营许可证 |
|---|:---:|:---:|
| 性质 | 登记/告知 | 正式经营许可 |
| 法律依据 | 《互联网信息服务管理办法》 | 同上 |
| 适用范围 | 所有网站/小程序 | 经营性互联网信息服务 |
| **本项目是否需要** | ✅ 必须 | ⚠️ 灰色地带 |
| 微信是否检查 | 是 | 深度合成类目不查 |
| 办理周期 | 约20天 | 40-60天 |
| 费用 | 免费 | 代办约3-5k |

#### 为什么ICP经营许可证是"灰色地带"？

- **法律层面**：提供付费信息服务（会员订阅）属于经营性，理论上需ICP许可证
- **微信层面**：深度合成-AI问答类目审核不要求提供ICP许可证
- **行业实际**：大量带会员付费的小程序未持有ICP许可证正常运营中
- **建议**：先上线，边跑边办，不卡上线节奏

### 2.3 微信支付 / 虚拟支付

#### 安卓端：微信虚拟支付
- 接入微信支付商户号，开通小程序虚拟支付
- 手续费约0.6%

#### iOS端：Apple订阅通道
- 走 `wx.requestAppleSubscribeSign` 接口
- Apple佣金：**12%**
- 准入条件：
  - 已接入小程序虚拟支付
  - 小程序首次发布 ≥ 90天
  - 近30天日均DAU ≥ 1万
- 新上线小程序前期可能达不到DAU门槛，需要规划

## 三、小程序 vs 服务号H5 对比

| | 微信小程序 | 服务号内嵌H5 |
|---|:---:|:---:|
| **微信AI类目审核** | 必须，会检查 | 不审核 |
| **算法备案材料** | 必须上传 | 不检查（法律上仍需） |
| **iOS支付** | 走Apple订阅，抽成12% | 走微信JSAPI，无Apple抽成 |
| **用户留存** | 好，有桌面快捷+下拉入口+模板消息 | 弱，依赖菜单和推文 |
| **分享传播** | 强 | 弱 |
| **开发成本** | 原生或跨端框架 | 纯web，成本低 |
| **适合阶段** | 正式上线运营 | 快速验证MVP |

> 💡 **建议**：追求快速验证可先上服务号H5（免Apple税），验证通过后补小程序。
